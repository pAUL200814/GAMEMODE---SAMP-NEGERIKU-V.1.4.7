Perubahan yang dilakukan meliputi:

Seluruh dialog di-remake.
Sebagian besar command (CMD) ditulis ulang dan disesuaikan.
Sistem job dikembangkan ulang.
Loading screen dibuat ulang.
fixed bug
Berbagai sistem inti gamemode dirombak dan ditingkatkan.
etc

Repository ini disimpan sebagai arsip sekaligus monumen sejarah bagi gw dan para pengembang terdahulu negeriku.
